StellarCheckout

Methods

(inner) render(selector, opts)

Renders the default user interface.

Parameters:
Name Type Description
selector string

The selector of the target element where StellarCheckout will be added to the DOM.
opts object

A configuration object.

Properties
Name Type Description
culture string

Options: ['en', 'es-ES', 'ja-JP', 'ko-KR', 'ru-RU', 'zh-CN']
currency string

Options: ["AUD", "BRL", "CAD", "CHF", "CLP", "CNY", "CZK", "DKK", "EUR", "GBP", "HKD", "HUF", "IDR", "ILS", "INR", "JPY", "KRW", "MXN", "MYR", "NOK", "NZD", "PHP", "PKR", "PLN", "RUB", "SEK", "SGD", "THB", "TRY", "TWD", "ZAR"]
from string

The sender's address. A federated stellar address OR a public key.
id string

An optional reference ID supplied by the merchant
network string

The Stellar.org network. Options: [ 'public', 'test']
onSubmit callback

A callback function that will contain the transaction results. Uses error first callback pattern.
redirectUrl string

A URL that the user will be redirected to upon completing a successful transaction.
showNetworkSelector boolean

Shows the network selector component. Options: [true, false]
showTicker boolean

Show the XLM ticker component Options: [true, false]
stylesheet string

Allows overriding of the default styles. Must be a valid URL to a stylesheet document.
to string

The recipient's address. A federated stellar address OR a public key.
total decimal

The invoice/order total in the fiat currency specified in opts.currency

Basic Configuration Example
